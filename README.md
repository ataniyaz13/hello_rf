# Environment setup

## Create virtual environment for tests execution
```bash
Ubuntu:
cd hello_rf
python3 -m venv venv && source venv/bin/activate
pip install -r requirements.txt

Windows:
cd hello_rf
python -m venv venv && source .\venv\Scripts\activate.bat
pip install -r requirements.txt
```
